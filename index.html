mixup!

mixup! is a set of sass mixins coming with default styles. It is designed to allow you to quickly integrate webdesigns with DRYer styles.

There are three mixin families:

  • style.scss: the low level styles (borders, backgrounds, gradients...).
  • component.scss: ready to skin components (buttons, tables, forms...).
  • layout.scss: the layout builders (grids, containers...).

Installation

{% highlight bash %} gem install sass git clone git@github.com:jeanphix/mixup.git {% endhighlight %}

Sample usage

{% highlight css %} @import "mixup/mixin/default/all"; @import "mixup/mixin/component"; a { @include button-simple(); } {% endhighlight %}
