mixup!

mixup is a set of sass mixins that helps you to build your web designs with DRYer stylesheets.

$ gem install sass
$ git clone git@github.com:jeanphix/mixup.git

mixins are split into three files:

  • style.scss: the low level styles (borders, backgrounds, gradients...).
  • component.scss: ready to skin components (buttons, tables, forms...).
  • layout.scss: the layout builders (grids, containers...).
